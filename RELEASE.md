# Withered Blade - Release Distribution

This folder contains the compiled Minecraft mod ready for distribution.

## Installation

1. Download `witherd_blade-release.jar` from the Releases page
2. Place it in your Minecraft Forge mods folder: `%appdata%\.minecraft\mods\`
3. Launch Minecraft with Forge

## Requirements

- **Minecraft:** 1.20.1
- **Forge:** Latest version
- **Dependency:** Cracker's Wither Storm Mod (v4.2.1)
