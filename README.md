**Withered Blade** is a powerful Forge 1.20.1 mod that adds the ultimate weapon forged from the Wither Storm’s heart.  
Craftable only with rare materials from Cracker's Wither Storm mod, this sword is the definition of destruction.

---

## 🔥 Features

- ⚔️ A sword with **3 billion attack damage**
- 📦 Clean, optimized, and survival-compatible

## Recipe 
         **Witherd Nether Star**
         **Stick**

         supports player and crafting table slots

## 🗒️ Requirements

 - [Cracker's Wither Storm v4.2.1](https://modrinth.com/mod/crackers-wither-storm-mod/version/4.2.1)
