**Withered Blade** is an advanced weapon mod for Minecraft Forge 1.20.1. It introduces the ultimate late-game weapon, forged directly from the core of the Wither Storm. Craftable exclusively using rare, end-game materials harvested from Cracker's Wither Storm Mod, this blade represents the absolute pinnacle of destructive power.

---

## Features

* **Unrivaled Power:** Deploys a massive **3,000 attack damage** to slice through the toughest foes.
* **Highly Optimized:** Clean code structure ensures seamless integration and performance.
* **Survival Ready:** Fully integrated into survival gameplay as a legitimate, high-tier crafting objective.
* **Secret Ability:** Features a hidden mechanic waiting to be discovered by worthy players.

Tip: Hold **Withered Blade** in your offhand with a Netherite Pickaxe in your hand with Efficiency 5 to insta-mine Stone
